======

Панель управления ssh серверами

Возможности
------

* Просмотр информации о дисках
* Работа в консоли
* Файловый менеджер
* Авторизация и сохранение серверов

Запуск
------

### Запуск на **Linux**
* В папке proxy
  ```bash
  go run ./proxy_server.go
  ```
* Запуск в фоне с логированием в файлы:
  ```bash
  go run ./proxy_client.go
